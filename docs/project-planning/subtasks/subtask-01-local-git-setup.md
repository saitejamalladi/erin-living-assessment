# Subtask 1: Local Git Setup

## Description
Initialize a local Git repository for the project, configure basic Git settings (user name, email), create a main branch, and add an initial .gitignore file suitable for a Node.js/NestJS project. This sets up version control for the entire development process.

## Tasks
- Initialize Git repository in the project root.
- Configure global Git user name and email.
- Create and switch to main branch.
- Add .gitignore with entries for Node.js (node_modules, dist), Docker (volumes), logs, and environment files (.env).
- Make initial commit with the .gitignore file.

## Git Workflow
- Create a new feature branch: `git checkout -b feature/subtask-01-local-git-setup`
- Commit changes with brief messages, e.g.:
  - "Init Git repo"
  - "Add .gitignore"
- Before raising PR: No unit tests or build yet (no code), but ensure repository is clean.
- Raise PR to main with title "Add local Git setup" and description "Initialize Git repository with basic config and .gitignore."</content>
